# PortFolioMaker

## Description
PortFolioMaker is a free tool to help you create stunning portfolios easily. Currently under development, this project aims to provide a React-based application for building professional portfolios with minimal effort.

## Features
- Easy-to-use interface
- Customizable templates
- Responsive design
- Free to use

## Getting Started
1. Clone the repository: `git clone https://github.com/ghoshswapnadip7-coder/PortFolioMaker.git`
2. Open `index.html` in your browser to view the intro page.
3. For full functionality, the React application is in development.
